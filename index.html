Web Apps

Experience

A social media application that lets users share images of their current activities and like that of other users. Experience was made using React and Material-UI to create the front-end, redux to handle state management across the entire application, GoogleAuth and JWT for user authentication and authorization, MongoDB for persistent data storage, Express and Node.js to create the back-end API.

The Shoppies

This is a web application that utilizes the OMDb API. It fetches movie data from the API and then renders it in a front-end built with react and the Shopify-Polaris component library. The application lets users nominate and withdraw the nomination of their favorite films. The context API was used to manage the react states in this application.
